interface Callback : PillarboxMediaSession.Callback

An extended PillarboxMediaSession.Callback for the PillarboxMediaLibrarySession.

When you return LibraryResult with MediaItem media items, each item must have valid mediaId and specify isBrowsable and isPlayable in its mediaMetadata.
