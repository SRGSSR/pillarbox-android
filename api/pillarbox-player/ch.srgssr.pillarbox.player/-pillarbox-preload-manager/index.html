PillarboxPreloadManager

class PillarboxPreloadManager(context: Context, targetPreloadStatusControl: TargetPreloadStatusControl<Int, DefaultPreloadManager.PreloadStatus>? = null, mediaSourceFactory: PillarboxMediaSourceFactory = PillarboxMediaSourceFactory(context), trackSelector: TrackSelector = PillarboxTrackSelector(context), bandwidthMeter: BandwidthMeter = PillarboxBandwidthMeter(context), renderersFactory: RenderersFactory = PillarboxRenderersFactory(context), loadControl: LoadControl = PillarboxLoadControl(), playbackThread: HandlerThread = HandlerThread("PillarboxPreloadManager:Playback", Process.THREAD_PRIORITY_AUDIO))(source)

Helper class for Media3's DefaultPreloadManager that simplifies preloading of media items for PillarboxExoPlayer.
