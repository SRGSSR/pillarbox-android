fun Modifier.toggleable(enabled: Boolean = true, role: Role? = Role.Switch, indication: Indication? = null, interactionSource: MutableInteractionSource?, delayedVisibilityState: DelayedVisibilityState): Modifier

Makes a Composable toggleable, controlling the visibility state of a DelayedVisibilityState.

Parameters

enabled

Whether to handle input events and appear enabled for semantics purposes.

role

The type of UI element. Accessibility services might use this to describe the element.

indication

Indication to be shown when the Composable is pressed. If null, no indication will be shown.

interactionSource

The MutableInteractionSource that will be used to dispatch Interactions when this toggleable component is being interacted with.

delayedVisibilityState

The DelayedVisibilityState instance to control the visibility of the component.
