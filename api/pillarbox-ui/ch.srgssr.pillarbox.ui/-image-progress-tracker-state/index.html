ImageProgressTrackerState

class ImageProgressTrackerState(player: PillarboxPlayer, coroutineScope: CoroutineScope, imageOutput: ImageOutput) : ProgressTrackerState(source)

A ProgressTrackerState implementation that updates the Player progress every time onChanged is called and set the player to render the image track into the imageOutput.

Parameters

player

The Player whose progress needs to be tracked.

coroutineScope

The CoroutineScope used for managing StateFlows.

imageOutput

The ImageOutput to render the image track.
